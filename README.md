# Interactive Calendar Component

A polished, responsive web component inspired by physical wall calendars. This project was built to demonstrate advanced frontend engineering skills in React, including complex state management, custom styling, and responsive layout design.

![Calendar Preview](https://github.com/Shashankm091/Interactive-Calendar/raw/main/public/assets/hero.png)

## ✨ Features

- **Wall Calendar Aesthetic**: Cinematic hero imagery with a clean, physical-inspired date grid.
- **Day Range Selector**: Interactive range selection with visual highlighting for start, end, and intermediate dates.
- **Persistent Notes**: Integrated monthly memos that save automatically to `localStorage`.
- **Today Highlight**: Real-time identification and highlighting of the current date.
- **Fully Responsive**: Optimized for both desktop (side-by-side) and mobile (stacked) viewports.
- **Modern Tech Stack**: Built with React, Vite, and date-fns for optimal performance.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (v7 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Shashankm091/Interactive-Calendar.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Interactive-Calendar
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **React**: Core library for UI.
- **Vite**: Ultra-fast build tool and dev server.
- **date-fns**: Modular date utility library.
- **Lucide React**: Beautiful & consistent icons.
- **Vanilla CSS**: Custom design system for maximum flexibility.
